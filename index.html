Decision execution framework

6-step flow for every flagged physical threat

This is the decision sequence operators run every shift. It turns alerts into accountable outcomes.

3D presentation flow: each stage is a decision gate you can scan in one pass.

1. Detect

Candidate flagging

AI identifies threats in camera and thermal streams and assigns confidence.

2. Assign

Named ownership

Each incident is linked to one responsible user and a required outcome.

3. Acknowledge

Human validation

No closure without operator acknowledgment.

4. Resolve / Escalate

Actioned response

Choose False Positive, resolve on site, escalate to patrol, or verify by drone.

5. Audit

Timestamped actions

Resolution events are logged with who/when/decision notes.

6. Report

KPI and trend review

Performance is checked on acknowledge time, evidence ratio, and unresolved locations.
