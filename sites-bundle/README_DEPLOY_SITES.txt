# Sites Deployment Bundle

This bundle includes both proposal prototypes:
- index.html
- boustead_proposal_story_mode.html

To publish to Sites (static hosting):
1) Open each Files tab in your Sites project.
2) Upload both HTML files into the same directory / root.
3) Set the published routes as:
   - /index.html
   - /boustead_proposal_story_mode.html
4) (Optional) rename one file to "index.html" for default home path if needed.
